# Price Drop Tracker (Amazon + Flipkart)

Automatically tracks product prices and sends email alerts when price drops.

## Features
- Amazon + Flipkart support
- Stores price history
- Tracks lowest, highest, average price
- Runs automatically every 2 hours
- Email alerts using Gmail SMTP
